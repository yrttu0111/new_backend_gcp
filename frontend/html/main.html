GraphQL은 REST API의 단점을 보완하고자 만들어진 API입니다.

  아래 버튼을 눌러 시험하실 수 있습니다.

그외 기능들

restAPI 로 만들어진 다른 기능들입니다.
결제기능은 아임포트를 이용해 만들었습니다.(결제시 밤에 자동 환불)
결제를 하면 계정에 포인트가 쌓이게 만들었습니다. 
(환불 등은 graphQL에서 테스트 가능)

로그인은 소셜로그인(구글 네이버 카카오)과 일반로그인을 구현했습니다.
소셜로그인 기능은 passport를 이용해 만들었습니다. 
소셜로그인 시 처음 로그인 시에는 회원가입을 하고
그 후에는 로그인을 합니다.
로그인은 JWT를 이용해 만들었습니다.
로그아웃은 redis를 이용해 만들었습니다.
백엔드 프로젝트이다 보니 회원가입 페이지를 만들지 않았지만 
graphql playground에서 회원가입을 할 수 있습니다.

이미지 업로드는 gcp bucket을 이용해 만들었습니다.
-- 과금으로 인해 링크를 없애놓았습니다.
포인트 충전 로그인

배포


배포는 GCP kubernetes를 이용해 만들었습니다.
database는 GCP sql을 이용해 만들었습니다.
이미지 관리는 GCP bucket을 이용해 만들었습니다.
cloud funcion을 이용해 이미지 업로드시에
bucket에 이미지를 업로드하고
database에 이미지 url을 저장합니다.
CI/CD 로 깃허브와 연결해 무중단 배포를 구현했습니다.
